🥗 NutriTodoSalud - Analizador de Comidas con IA

NutriTodoSalud es una aplicación web progresiva (PWA) diseñada para ayudar a las personas a llevar un control nutricional de forma sencilla y visual. Utilizando la potencia de la Inteligencia Artificial (Gemini 2.5 Flash), la app analiza fotos de platos de comida para estimar macronutrientes y calorías al instante.

✨ Características Principales

📸 Análisis por Imagen: Toma una foto o sube una imagen de tu plato y deja que la IA identifique los alimentos.

🔢 Cálculo de Macronutrientes: Obtén estimaciones de:

Calorías Totales (kcal).

Proteínas (g).

Carbohidratos (g).

Grasas (g).

📋 Desglose Detallado: Identificación individual de cada ingrediente detectado en el plato.

🕒 Historial Diario: Registro local de tus comidas del día para un seguimiento rápido.

📱 Experiencia Móvil: Interfaz optimizada para dispositivos Android y iOS.

🔒 Privacidad: El historial se guarda localmente en tu navegador.

🚀 Cómo empezar

Requisitos

Un navegador web moderno (Chrome, Safari, Edge).

Conexión a internet.

Instalación en PC

Clona este repositorio o descarga el archivo index.html.

Abre el archivo en tu navegador favorito.

Instalación en Android (Como App)

Accede a la URL donde tengas alojado el proyecto (ej. GitHub Pages).

Toca el icono de tres puntos (menú) de Chrome.

Selecciona "Instalar aplicación" o "Añadir a la pantalla de inicio".

¡Listo! Ahora tendrás el acceso directo en tu menú de aplicaciones.

🛠️ Tecnologías utilizadas

HTML5 & CSS3: Estructura y diseño.

Tailwind CSS: Estilizado moderno y responsivo.

JavaScript (ES6+): Lógica de la aplicación y manejo de cámara.

Gemini 2.5 Flash API: El motor de IA encargado del reconocimiento de imágenes y análisis nutricional.

FontAwesome: Iconografía.

📝 Nota sobre la precisión

NutriTodoSalud proporciona estimaciones basadas en análisis visuales de IA. Para una precisión médica o dietética estricta, siempre se recomienda consultar con un profesional de la salud o nutricionista.
